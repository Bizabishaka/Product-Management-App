# My Product-Management App

A Vue 3 + TypeScript + TailwindCSS admin dashboard with sidebar, navbar, and multiple layouts for products, orders, and customers.

---

## Features

- Responsive sidebar and navbar.
- Mobile-friendly sidebar overlay.
- Product management (list, view, add, edit, delete).
- Orders and customers pages.
- Pinia state management.
- Vue Router with multiple layouts.
- Authentication guard.
